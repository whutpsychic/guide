##  1  获取全部角色

返回一个数组，包含以下字段

| 字段     | 类型     | 含义   | 备注   |
| ------ | ------ | ---- | ---- |
| roleId | int    | 角色Id |      |
| name   | string | 角色名  |      |



##  2  获取指定角色的权限

请求数据

| 字段     | 类型   | 含义   | 备注   |
| ------ | ---- | ---- | ---- |
| roleId | int  | 角色Id |      |

返回一个数组，包含以下字段

| 字段         | 类型     | 含义   | 备注                                      |
| ---------- | ------ | ---- | --------------------------------------- |
| menuId     | int    | 菜单Id |                                         |
| hasFeature | string | 功能点  | 角色在菜单中拥有的功能，使用逗号分隔多个功能点英文名。例如`add,edit` |



##  3  修改指定角色的权限

提交的内容包含两部分

第一部分是要修改的角色Id

| 字段     | 类型   | 含义   | 备注   |
| ------ | ---- | ---- | ---- |
| roleId | int  | 角色Id |      |

第二部分是修改后的该角色权限信息。权限信息的格式是一个对象数组，数组每一项字段如下：

| 字段         | 类型    | 含义   | 备注   |
| ---------- | ----- | ---- | ---- |
| menuId     | int   | 菜单Id |      |
| hasFeature | sting | 功能点  |      |
