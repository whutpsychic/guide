##  1  获取全部机构

返回一个数组，包含以下字段

| 字段       | 类型     | 含义    | 备注              |
| -------- | ------ | ----- | --------------- |
| deptId   | int    | 机构Id  |                 |
| name     | string | 机构名   |                 |
| parentId | int    | 父机构Id | 特殊的，一级机构该字段取值为0 |
