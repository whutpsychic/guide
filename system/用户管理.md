##  1  获取全部角色

返回一个数组，包含以下字段

| 字段     | 类型     | 含义   | 备注   |
| ------ | ------ | ---- | ---- |
| roleId | int    | 角色Id |      |
| name   | string | 角色名  |      |



##  2  查询用户列表

请求数据

| 字段       | 类型     | 含义   | 备注          |
| -------- | ------ | ---- | ----------- |
| name     | string | 用户名  | 根据用户名进行模糊查询 |
| page     | int    | 页码   | 从1开始，1表示第一页 |
| pageSize | int    | 每页条数 |             |

返回一个数组，包含以下字段

| 字段     | 类型     | 含义   | 备注             |
| ------ | ------ | ---- | -------------- |
| userId | int    | 用户Id |                |
| name   | string | 用户名  | 登录时使用的用户名就是该字段 |
| roleId | int    | 角色Id | 描述该用户属于的角色     |
